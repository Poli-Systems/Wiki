# Qu'est-ce qu'un parking ?

Si vous avez acheté un domaine et souhaitez le conserver sans publier de site complet, vous pouvez afficher une page indiquant que le domaine est garé et vous appartient.

Le parking aide aussi à préserver le référencement pendant les interruptions en affichant une page de maintenance plutôt qu'un résultat vide.

Si vous souhaitez vendre votre domaine, une page de parking rassure les acheteurs sur le fait que le domaine est actif.

Chez Poli Systems, nous proposons cette solution gratuitement afin que votre domaine reste en ligne avec une simple page d'introduction.

# Comment configurer votre domaine

## Avec n'importe quel fournisseur de DNS

La configuration est simple : il suffit d'ajouter deux enregistrements DNS (ou un seul si vous ne souhaitez qu'IPv4).

- `@` ou `*` pointant vers **116.203.149.108** en enregistrement **A**
- `@` ou `*` pointant vers **2a01:4f8:c2c:21e1:0000:0000:0001** en enregistrement **AAAA** (en version courte : **2a01:4f8:c2c:21e1::1**)
