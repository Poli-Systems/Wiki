# Ajouter un sous-domaine à votre serveur de jeu

Avec votre serveur de jeu, vous pouvez associer un sous-domaine personnalisé correspondant au nom de votre instance.

Cela vous permet de rejoindre votre serveur via une adresse simple et propre, comme **mygameserver.poli.games**.

Vous pouvez créer et gérer vos sous-domaines directement depuis votre panneau de gestion.

Commencez par vous rendre dans le **Gestionnaire de sous-domaines**.

![Gestionnaire](https://i.imgur.com/8637BAp.png)

Une fois dessus, cliquez sur **Create**.

![Bouton Create](https://i.imgur.com/zm0o6ZF.png)

Choisissez ensuite le sous-domaine que vous souhaitez utiliser pour votre serveur.

![Input](https://i.imgur.com/YsD0xhx.png)

Une fois le sous-domaine créé, vous pourrez rejoindre votre serveur via une adresse comme : **mygameserver.poli.games**.

Le principal avantage de cette méthode—particulièrement pour les serveurs Minecraft—est que vous n’aurez plus besoin d’indiquer le port. Cela offre une connexion plus simple, plus propre et plus professionnelle.
