# Recréation d’une sauvegarde de jeu

Un changement de version Factorio peut corrompre ou désynchroniser une sauvegarde. Pour la remettre en état :

1. Ouvrez et réenregistrez la partie avec votre client Factorio dans la version cible, puis téléversez cette sauvegarde sur le serveur.
2. Connectez-vous sur [https://game.polisystems.ch](https://game.polisystems.ch) (ou via l’espace client) et cliquez sur **Manage**.

   ![](https://i.imgur.com/8rBO93N.png)

3. Ouvrez le **File Manager**.

   ![](https://i.imgur.com/F5kecjV.png)

4. Supprimez le `gamesave.zip` existant.

   ![](https://i.imgur.com/F9Khxa7.png)

5. Redémarrez le serveur : la sauvegarde recréée devrait se charger normalement.
