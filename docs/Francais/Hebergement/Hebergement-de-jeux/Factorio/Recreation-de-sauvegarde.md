# Recréation d’une sauvegarde de jeu

-   Parfois, vous pouvez avoir ce problème lors du changement de version de Factorio.  
    Si vous souhaitez "convertir" une ancienne sauvegarde, utilisez votre client Factorio, puis réuploadez la sauvegarde sur le serveur.
-   Connectez-vous à votre panel sur https://game.polisystems.ch ou via votre espace client  
    (si vous n’avez pas encore configuré votre compte, vérifiez vos e-mails).
-   Une fois connecté, cliquez sur **manage**.

![](/mnt/data/c9d587dc-630b-449c-ab18-00fcc222172b.png)

-   Ici, vous aurez une vue complète de votre serveur Factorio.

![](https://i.imgur.com/Cxys6zW.png)

-   Une fois ici, allez dans le **File Manager**.

![](https://i.imgur.com/F5kecjV.png)

-   Supprimez votre fichier **gamesave.zip**.

![](https://i.imgur.com/F9Khxa7.png)

-   Redémarrez votre serveur, et tout devrait fonctionner !
